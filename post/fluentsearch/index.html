Fluent Search - 可选择的Listary替代品

2022-08-01 2 min read

使用 Fluent Search，你可以搜索正在运行的应用程序、浏览器标签、应用内容、文件等等。

特性

进程搜索

进程搜索可以让你轻松地找到你电脑上的任何活动进程。
你可以通过搜索窗口的标题或进程的名称来找到它。
另外，Fluent Search可以对应用内的元素进行查找，如标签、按钮甚至是进程内的超链接，以便立即访问它们。

屏幕搜索

有些人发现使用鼠标会降低工作效率，特别是在编程等非常注重键盘的工作中。
如果你是这些人中的一员，Fluent Search为你提供了保障。屏幕搜索功能可以让你用键盘轻松控制你的电脑。

搜索标签

如上所述，流畅搜索在你的电脑中搜索各种资源，这包括但不限于文件、应用程序、进程、书签、历史结果等等。
为了帮助你轻松获得最相关的结果，Fluent Search 有一个叫做 "搜索标签 "的功能。
Fluent Search 中的每个结果都可能包含描述该结果的多个标签，使用这些标签，你可以轻松地过滤你的搜索，找到最相关的结果。

网络搜索

觉得需要立即在网络上进行内容搜索，而不需要打开浏览器并创建一个新的选项卡？
那就对了，使用 Fluent Search，你可以轻松地在你最喜欢的搜索引擎中搜索。

定制化

流利搜索是以现代设计为基础的，但最重要的是，它能够让你自定义。
你是极简主义风格的粉丝？你想在屏幕上获得尽可能多的信息？流利搜索为你提供了保障。你可以轻松地缩放搜索窗口，并根据你的需要改变其外观。

导航

简洁性是流畅搜索非常重要的方面。
试图在提供丰富的搜索结果的同时保持简单，是 Fluent Search 一直面临的挑战。
